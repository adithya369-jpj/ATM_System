## Description
A console-based ATM system built in Python with **SQLite database integration**.  
Users can login, deposit, withdraw, transfer money, and view transaction history.  
This project demonstrates **OOP principles, database management, and menu-driven applications**.

## Features
- User login with account number and PIN  
- Deposit and withdrawal functionality  
- Transfer money between accounts  
- Mini statement showing last 5 transactions  
- Transaction logging in SQLite database  
- Sample accounts included for testing  

## Tech Stack
- Python 3.x  
- SQLite
